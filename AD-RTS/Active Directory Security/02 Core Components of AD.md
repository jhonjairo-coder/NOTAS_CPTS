- Domain Controller (DC) : Server running AD DS, stores the directory database. 
- Schema : Defines all object types and their attributes.
- Global Catalog : Subset of AD attributes for fast, forest-wide searches.
- Database (NTDS.dit) : The core file storing all AD objects and their attributes.
