#### What is Active Directory?

- Microsoft's directory service for Windows domain networks. 
- Centralized management of network resources (users, computers, printers, applications). 
- A phone book and security guard for your entire network.

![[Pasted image 20251106125901.png]]

#### Why Active Directory?

- Centralized Authentication : Single Sign on for users 
- Centralized Authorization : Granular Access control to resources 
- Centralized Management : Easier Administrations of users, devices, policies
- Scalability & Reliability : Distributed Architecture
